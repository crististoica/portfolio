Quiz App for CS Students

An app I made to help me and my colleagues prepare for the final year exam. The exam was required in order to be able to present the final project.

Made with:

  • ReactJS
  • Redux
  • MATERIAL - UI
  • NodeJS
  • Express
  • MongoDB

Main features:

  • Quizzes with random questions for the three required subjects: Java, Databases, Networks.
  • Community: after taking a quiz, an option to create a post will appear next to each question; so, students that were less prepared for that subject, could ask for help.
  • An admin panel used by admins to do CRUD (create, read, update, delete) operations on quizzes and community.
